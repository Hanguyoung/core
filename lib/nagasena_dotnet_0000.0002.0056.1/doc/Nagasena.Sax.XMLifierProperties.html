XMLifier Properties

The properties of the XMLifier class are listed below. For a complete list of XMLifier class members, see the XMLifier Members topic.

Public Instance Properties

AlignmentType Set the bit alignment style used to compile the EXI input stream.
GrammarCache Set the GrammarCache used in parsing EXI streams.
PreserveLexicalValues Set to true if the EXI input stream was compiled with the Preserve Lexical Values set to true. The original strings, rather than logical XML equivalents, are restored in the XML output stream.
