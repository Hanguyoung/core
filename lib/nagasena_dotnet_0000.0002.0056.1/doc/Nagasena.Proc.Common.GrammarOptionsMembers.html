GrammarOptions overview

Public Static Fields

DEFAULT_OPTIONS Indicates that undeclared elements and attributes will be processed when the XML stream is encoded and decoded.
OPTIONS_UNUSED OPTIONS_UNUSED is an internal value. It indicates that the grammar options value has not yet been set.
STRICT_OPTIONS Indicates that undeclared elements and attributes will throw an exception when the XML stream is encoded and decoded. When STRICT_OPTIONS is set, all other Grammar Options are ignored.

Public Static Methods

addDTD Sets Preserve Document Type Definition to true. (Adds 16 to the options value.)
addNS Sets Preserve Namespaces to true. (Adds 4 to the options value.)
addPI Sets Preserve Processing Instructions to true. (Adds 64 to the options value.)
